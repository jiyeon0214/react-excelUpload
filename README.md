** 프로젝트 소개
  엑셀 파일을 업로드하여 데이터를 JSON 형태로 변환하고, 웹에서 직접 수정할 수 있는 기능 구현
  특히 병합된 셀을 처리하여 데이터 손실 없이 테이블 형태로 변환 가능

** 프로젝트 목적 **
  - 엑셀 데이터 처리 방식 이해
  - 프론트엔드에서의 파일 처리 경험
  - 실무에서 자주 발생하는 병합 셀 문제 해결 경험

** 주요 기능
  - 엑셀 파일 업로드 (.xlsx, .xls)
  - 엑셀 데이터를 JSON으로 변환
  - 병합된 셀 자동 처리 (값 확장)
  - 테이블 형태로 데이터 수정
  - 수정된 데이터 JSON으로 저장

** 사용 기술
  - Frontend
    React (useState)
  - Library
    xlsx

** 동작 방식
1. 파일 업로드
FileReader를 사용하여 엑셀 파일을 ArrayBuffer로 읽음

2. 엑셀 파싱
xlsx.read()를 통해 Workbook 생성
첫 번째 시트를 기준으로 데이터 처리

3. 병합 셀 처리
worksheet['!merges'] 정보를 활용
병합 영역의 시작 셀 값을 모든 셀에 복사

4. JSON 변환
빈 셀은 ''로 처리
사용자 친화적인 값으로 변환

5. 데이터 편집
HTML 테이블로 렌더링
input을 통해 실시간 수정 가능


** 핵심 구현 포인트 **
1. 병합 셀 데이터 복원
엑셀에서 병합된 셀은 실제로 하나의 셀에만 값이 존재함
→ 이를 모든 셀에 복사하여 데이터 일관성 유지

2. 동적 테이블 생성
JSON 데이터 기반으로 헤더 자동 생성
유연한 데이터 구조 대응

3. 상태 관리
React의 useState를 활용한 데이터 관리
변경 시 즉시 UI 반영

** 느낀점
엑셀 데이터를 단순 변환하는 것을 넘어 실제 업무에서 발생할 수 있는 데이터 구조 문제(병합 셀)를 해결하는 경험을 할 수 있었다.
또한 파일 처리와 데이터 변환 로직을 직접 구현하면서 프론트엔드에서도 데이터 가공이 충분히 가능하다는 점을 배웠다.
